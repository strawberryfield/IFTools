# Command line usage for GBlorbEd 
``` 
Casasoft GBlorb command line editor v1.0
(c) 2025 Roberto Ceccarelli - Casasoft

Usage: GBlorbEd [OPTIONS] FILE -o OUTFILE
Edits components of a GBlorb file.

Options:
  -q, --quiet                Suppress banner print
  -h, -?, --help             Show this help
  -o, --output=VALUE         Output filename
  -f, --frontispice=VALUE    Frontispice resource number
  -a, --author=VALUE         Author name
  -c, --copyright=VALUE      Copyright notice
  -r, --remarks, --annotations=VALUE
                             Annotations
```
